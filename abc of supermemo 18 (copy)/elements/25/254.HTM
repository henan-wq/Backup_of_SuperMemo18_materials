2.3 渐进阅读的五项基本技能 : 批量导入

将学习材料导入SuperMemo最方便的方法是直接从web导入多篇文章。

要同时导入多篇文章，请在Internet Explorer中打开这些文章，然后单击learnbar上的Import按钮（或按Ctrl+Shift+A，或在主菜单上选择Edit >> Web import >> All）。

要避免导入广告和其他垃圾信息，请在Internet Explorer中选择要导入的文本部分。如果在导入之前选择文本，就不用通过筛选器（可在导入的文章上按F6键）来过滤掉不需要的HTML代码。

如果您喜欢使用其他浏览器，例如Chrome或Firefox，则需要使用复制粘贴方法，或者在Internet Explorer中重新打开所选文章才能导入。

SuperMemo目前只支持从Internet Explorer直接导入文章。

有关批量导入文章的详细信息，请参见：从Web导入*
