You can import many articles from Internet Explorer if you select Edit : Import web pages: All from the main menu* (alternatively, press Ctrl+Shift+A).

If you press Ctrl+Shift+W instead of Ctrl+Shift+A you will see only Wikipedia articles (instead of all articles).
Similarly, if you press Ctrl+Shift+Y, you will only see YouTube videos for import.

* If you cannot see Edit : Import web pages : All, select File : Level : Middle or higher.
