Title: 2.3 渐进阅读的五项基本技能


 

要想发挥渐进阅读最大的效能，需要数月甚至数年的练习。这个概述只会帮助你掌握基本技能，并帮助你开始使用渐进阅读。这五项基本技能是：

  • 将文章导入SuperMemo
  • 阅读文章并将文章分解成可管理的部分
  • 把重要的知识片段转化为问答材料
  • 复习材料以确保不会遗忘
  • 处理不可避免的信息大爆炸

技巧1：导入文章

最初，您可以通过简单的复制粘贴将文章导入SuperMemo。一段时间之后，您将需要掌握从web自动导入文章的功能，这样做有很多好处。

以下是5种向SuperMemo中导入文章的方法：

  • 复制粘贴：在浏览器（或任何允许复制文本的应用程序）中选择一篇文章的文本，将其复制到剪贴板，并用快捷键（Ctrl+N）将其复制到SuperMemo。
  • 批量导入：使用专用的web导入选项*从Internet Explorer导入大量文章。此方法将会避免导入重复文章、使用引用*标记文章、仅导入选定的文本部分，并且还有许多其他优点。
  • 专用导入：SuperMemo提供的工具使我们可以轻松的从Wikipedia和YouTube导入材料。Wikipedia是SM推荐的渐进阅读信息源；YouTube是推荐使用的渐进视频信息源。
  • 本地文件导入：导入已在硬盘上收集的文件
  • 邮件导入：用于邮件的渐进处理

复制粘贴导入

要通过复制和粘贴导入文章，请执行以下操作：

  • 在web浏览器中选择导入的文本并将所选内容复制到剪贴板（例如，使用Ctrl+C）
  • 切换到SuperMemo（例如，使用Alt+Tab键）
  • 在SuperMemo中，按Ctrl+N（或者通过菜单： Edit >> Add a new article ）。SuperMemo将创建一个新元素，并粘贴文章。您也可以使用“Read”工具栏上的“粘贴文章”按钮：
  • （可选）使用Alt+P定义导入文章的优先级。使用百分比字段设定优先级。记住0%是最高优先级，而100%是最低优先级。
  • （可选）使用Ctrl+J指定第一个复习间隔。例如：高优先级材料1天，低优先级材料30天。

记住，如果您在Internet Explorer中打开了许多文章，您可以按照下一节中的说明使用web导入来轻松导入它们。

批量导入

将学习材料导入SuperMemo最方便的方法是直接从web导入多篇文章。要同时导入多篇文章，请在Internet Explorer中打开这些文章，然后单击learnbar上的Import按钮（或按Ctrl+Shift+A，或在主菜单上选择Edit >> Web import >> All）。要避免导入广告和其他垃圾信息，请在Internet Explorer中选择要导入的文本部分。如果在导入之前选择文本，就不用通过筛选器（可在导入的文章上按F6键）来过滤掉不需要的HTML代码。如果您喜欢使用其他浏览器，例如Chrome或Firefox，则需要使用复制粘贴方法，或者在Internet Explorer中重新打开所选文章才能导入。这是因为，SuperMemo目前只支持从Internet Explorer直接导入文章。

有关批量导入文章的详细信息，请参见：从Web导入*

专用导入（维基百科、YouTube和图片）

最受欢迎的学习材料来源是Wikipedia（用于渐进阅读）和YouTube（用于渐进视频）。对于这些信息源，您可以在导入菜单上使用单独的选项（右键单击learnbar上的 Paste an article按钮（））。您还可以使用快捷键（Wikipedia：Ctrl+Shift+W）和（YouTube：Ctrl+Shift+Y）从这两个信息源导入。如果选择这些选项，则导入列表中将仅显示来自这些信息源的项目。此外，一个专用的筛选器将会把导入的页面处理好，以方便学习。

图：从维基百科导入一篇关于温室效应的文章（使用Edit>>Web import>>Wikipedia（Shift+Ctrl+W））后，它的整个文本都存储在一个单独的topic中。

如果要导入图片，还可以使用图片选择器，该选择器将忽略在Internet Explorer中打开的所有非图片页。使用（右键单击learnbar上的 Paste an article按钮（）），并选择Import pictures。导入页面时，也可以在“web导入”对话框（Ctrl+Shift+A）中单击Filter >> Pictures。

有关详细信息，请参见：从Web导入*

从本地文件导入项目

如果要导入本地驱动器上的文件，可以使用以下方法：

  • 从Internet Explorer导入一篇文章到新元素
    1.在Internet Explorer中打开本地文章
    2.导入方式与从web导入文章的方式相同（例如，通过复制粘贴或批量导入）
  • 将单个文章导入现有HTML组件
    1.在HTML组件菜单上选择File >> Import file
  • 多篇文章（存储在一个文件夹中）
    1.使用File >> Import >> Files and folders
    2.选择包含文章的文件夹（确保文件夹或其子文件夹中没有其他文件，否则，它们也将被导入SuperMemo）

技巧2：阅读文章

以下是阅读文章的简化方法：

  1. 选择一篇文章：按照前面的说明导入一篇文章，或者使用Learn（Ctrl+L）调出以前导入的文章。Learn将只显示过去导入的文章。如果要导入文章，并希望在同一天的学习会话中显示，则必须将其放入outstanding queue（在元素菜单上选择Learning >> Later today ，或者使用快捷键Ctrl+Shift+J）。如果要在同一天处理导入的全部项目，则必须将它们全部放入outstanding queue。例如，在浏览器中打开文章，然后选择Learning >> Add all to outstanding （或使用浏览器工具栏上的Add to outstanding图标）。大多数时候，你可以完全依赖Learn将文章安排在最佳的时间进行复习。
  2. 单击文章进入编辑模式，在编辑模式下，您可以修改文本、选择片段等。如果文本难以处理（例如片段无法选中、摘录片段未正确标记等），可以使用筛选器（F6）过滤掉多余的HTML代码。
  3. 从第一个或最后一个阅读节点开始阅读文章。
  4. 摘录文本：如果你在文章中遇到一个有趣的文本，选择它并在learnbar上选择Rememb extract（或按Alt+X）。此操作将把摘录的片段作为一篇独立的小文章引入学习过程。如果要指定新摘录片段的优先级，请选择Reading >> Schedule extract （，位于底部的Read工具栏上，可用于指定要摘录的文本片段的优先级），而不是Remember extract。另外，如果您觉得文章很难，并且希望以后再阅读一些片段，请使用Reading >> Schedule extract （）摘录这些片段，并根据您认为您能够更好地理解摘录的片段的时间而设置一个复习间隔。
  5. （可选）使用Delete before cursor（Alt+\）。这将删除您已阅读的文本、清理文章、删除垃圾并帮助处理难以处理的HTML。
  6. （可选）如果您读到一个看起来不重要的片段，请用鼠标选中并删除它（Del键）或用ignore样式标记它。若要将文本标记为ignore，请选择“组件”菜单上的 Reading >> Ignore，或者单击Read工具栏上的“忽略文本”按钮（/），或按Ctrl+Shift+I。
  7. （可选）如果所选片段不包含所有重要的上下文，则可以手动添加此上下文。例如，如果你正在学习历史，你可以从一篇关于林肯的文章中摘录以下片段：1862年9月22日，林肯总统发表了《解放宣言》，这是世界历史上重大事件之一。他在1863年1月1日签署了它。如果你想摘录‘他在1863年1月1日签署了它’的片段，你需要将他改为林肯，并将它改为《解放宣言》，这样你摘录的片段就变成了容易理解的：林肯于1863年1月1日签署了《解放宣言》。可以使用“组件”菜单上的Reference选项轻松地将上下文添加到摘录片段中（请参阅：引用*）。通过Reference添加的上下文将自动添加到给定文章的所有摘录中。例如，选择要用作所有摘录的引用标题的文本，然后在HTML组件菜单上选择 Reference >>Title（或按Alt+T）。此文本将显示在所有摘录的底部（默认情况下粉红色字体）。
  8. （可选）标记最后一个阅读节点：一旦决定在文章结束前停止阅读，将最后处理的片段标记为阅读节点（例如，使用Ctrl+F7或从HTML组件菜单中选择 Reading >>Read-points >> Set read-point）。下次你回到这篇文章时，SuperMemo会突出显示你的阅读节点，你将能够从上次停止阅读文章的那一点恢复阅读。要转到当前阅读节点，请按Alt+F7。如果忘记设置阅读节点，SuperMemo将在最后一个摘录处或最后一个突出显示的位置设置阅读节点。
  9. 转到下一篇文章：读完一篇文章的一部分后，选择 Learn 或者Next repetition继续阅读其他文章。这些按钮位于元素窗口的底部。您也可以使用Enter，只要片段在文本中不是无意义的（例如标记为阅读点），或者您已经退出编辑模式（例如使用Esc），Enter就可以跳转到下一篇文章。如果未选择文本，则Enter将在文本中添加新行（与标准文本编辑器的情况一样）。
  10. （可选）确定下一个复习日期（Ctrl+Shift+R），或者重新设置文章的优先级（Alt+P）。
  11. 在渐进阅读中，中断阅读是一种常规操作，而不是一场意外！通过一点练习，你会很快习惯这种不太自然的状态，并学会欣赏渐进方法的威力。中断的主要是防止阅读质量下降*。你可以使用以下标准来决定何时停止阅读文章：
    • 缺乏时间：如果你在一天内还有很多文章要复习，而你的时间已经不多了，那就缩短你的渐进。一段时间后，匆忙将成为一种常态，你将只阅读每篇文章的1-2个段落，只深入挖掘那些会对你的知识产生巨大影响的开创性文章。
    • 无聊：如果这篇文章让你觉得无聊，就停止阅读。你的注意力总是有限的。如果你的注意力不集中，那就休息一会儿再看这篇文章，这会让你从中受益更多。如果你还不感到疲惫，可以继续读一些其他的东西。如果你觉得SuperMemo安排的下一次复习太晚，请使用Ctrl+J或Shift+Ctrl+R来调整下一次复习日期。
    • 缺乏理解：如果你觉得在理解文章之前需要更多的知识储备，那么就推迟阅读（Ctrl+J或Shift+Ctrl+R，并在大约100天内安排下一次复习）。如果您已经导入了具有相关解释性知识的文章，则可以搜索这些文章（Ctrl+F）,找到它们后，可以（1）执行筛选组复习，或（2）将文章添加到outstanding queue中以便在同一天阅读，或（3）提前阅读文章（例如，在浏览器中，可以执行：Learning>>review all或Learning>>add to outstanding或Advance>>Topics）。如果您还没有导入任何解释性文章，现在就可以导入（例如，在网上搜索并向前面介绍的那样导入文章）。注意，您可以在SuperMemo中选择一段文本，并使用Ctrl+F3在百科全书或词典中搜索有关给定主题的更多材料。
    • 低优先级：将低优先级的文章分成小部分阅读，从而减少低优先级主题的总体时间分配。
    • 超负荷：如果你有一长串的文章要读，你确实只会对每篇文章读一小段。
  1. 一旦你读完了整篇文章并摘录了所有重要的片段，选择Done！（）。您也可以按Shift+Ctrl+Enter，然后在命令窗口中选择Done！，或在元素菜单上选择Learning >> Done 。Done！将关闭文章并删除其内容（不删除摘录的材料）。Done将删除一篇没有价值的文章（即没有提供任何摘录的文章）。使用Done将极大地减少您的收藏的大小，并在搜索文本时清除无效结果。

技巧3：摘录片段、问题和答案

摘录片段

在传统阅读过程中，我们经常用荧光笔标记重要段落。在SuperMemo中，这些段落可以作为单独的小文章摘录出来，稍后将用于唤醒您的记忆。每一个摘录的段落或部分都成为一个新的元素，它将遵循与原始文章相同的阅读算法。使用Extract（Alt+X，或者learnbar上的按钮）摘录重要片段和句子。

添加引用

在渐进阅读中，您总是需要快速回忆问题或文本的上下文。快速恢复上下文的最简单方法是添加引用。在摘录文本和制作完形填空时，引用会从一个元素传递到另一个元素。由于所有子级元素都是从同一个带有引用标记的给定文本生成的，因此您不必担心丢失上下文的问题。从web导入时，将自动添加引用。也可以逐个字段手动添加它们的引用。下面的图片示例中，引用以粉色文字显示。有关详细信息，请参见：引用*。

图：渐进阅读的典型截图。在学习温室效应的同时，学生摘录出片段：“在没有温室效应和大气层的情况下，地球表面平均温度14°C（57°F）可能低至-18°C（-0.4°F），即地球的黑体温度。”。摘录的片段将继承右边的插图以及文章引用。学生可以按回车键继续阅读另一篇文章。右边的图片存储在本地的图片注册表中（在用户的硬盘上），可以在其他元素中重复使用。

完形填空：生成问题

SuperMemo将向您展示摘录的重要片段并在以后的时间进行复习，将对您的记忆能力产生极好的影响。然而，它也会暴露出：一旦复习间隔时间超过200-300天，阅读和重读（被动复习）往往会导致回忆不足。因此，你迟早会需要把你的文本转换成具体的问题。为此你需要将摘录的文本转换成完形填空。完形填空是使用省略号（[…]）替换句子的一部分的问答卡片。例如： 问：塞拉利昂首都是[…] 答：弗里敦在渐进阅读中，完形填空是由句子或简单段落形式的摘录卡片生成的。要创建完形填空，请执行以下操作：

  1. 确保主题只包含一个简短的句子（例如，塞拉利昂首都是弗里敦）
  2. 在句子中选择一个重要的关键字（例如，弗里敦）
  3. 执行下列操作之一：

单击Read工具栏上的“完形填空”按钮（），或

按Alt+Z，或

从HTML组件菜单中选择Reading >> Remember cloze

Remember cloze会把一个句子转换成一个有答案的特定问题。通过使用完形填空，你将从被动复习转向主动回忆。在复习中，你不需要等一个段落或句子变得难以回忆再创建完形填空。对于重要的材料，你可以在记住你需要记住的信息后立即为此创建完形填空问答卡片。

简化问题

在将摘录转换成问答的同时，你应该确保你的问题简单明了，并带有相关的上下文。例如，如果您已从有关Internet历史的阅读中摘录了以下片段：

互联网是在1969年由高级研究计划局（ARPA）签订的一份合同下建立的，该公司连接了美国西南部大学（UCLA、斯坦福研究所、UCSB和犹他大学）的四台主要计算机

你可能会发现，当复习间隔变得足够长时，你可能实际上无法回忆起ARPA机构的名称，甚至忘记互联网开始的年份。然后，您可以选择一个重要的关键字，例如1969，并使用Remember cloze生成以下问答：

问题：互联网是在[…] 年根据高级研究计划署（ARPA）的合同开始的，该机构将美国西南部大学（UCLA、斯坦福研究所、UCSB和犹他大学）的四台主要计算机连接起来。

答案：1969年

在学习过程中，您需要通过手动编辑将上述问答卡片修改成更简洁易懂的形式：

问题：互联网是根据ARPA代理的一份合同于[…] 年启动的

答案：1969年

或者更好：

问题：互联网始于[…] 年

答案：1969年

至于编辑过程中“丢失”的宝贵信息，可以（但不必）通过Remember cloze产生的单独问题进行独立学习。

上述对问题的微调为新创建的问答带来了以下好处：

  1. 明确了问题的关键点：这个问题是关于互联网开始的年份的，这一点通过使用红色的（年份） （高亮）来强调。（抱歉，这里面显示不出高亮效果，可以在软件中设置高亮）
  2. 简洁：通过删除多余的信息，您不会浪费时间在不太可能被记住的信息上（只有主动回忆的材料才会被长期记住）。你将回答这个问题，永远不会关注哪些大学最初是通过早期的互联网连接的。如果你认为这些信息也很重要，你将使用原始的摘录来生成更多的以大学名称作为答案的完形填空问答卡片。（如果你不同意，请阅读：制定学习材料的20条规则）。
  3. 便于理解：“ARPA代理”这个短语可能会违反你在小学学到的语法规则，但它远比ARPA更容易理解。在SuperMemo中，容易理解比严格的语法或拼写规则更重要！

技巧4：重复和复习

SuperMemo 是基于重复复习理论的。你将经常复习所学的材料，以确保你不会忘记。

如果你以前从未尝试过SuperMemo，你需要掌握这里描述的标准复习技巧。

在渐进阅读中，你的复习将基于与传统SuperMemo相似的原则。主要区别在于：

  • 学习过程将把阅读新文章和复习你的文章结合起来
  • 你通常会有一完形填空形式的问答卡片，即句子中有一个缺失部分（[…] ）用于提出问题（例如，离太阳最近的行星是[…] ）。
  • 由于整个学习过程是渐进的，你的完形填空常常会以未完成的形式出现

渐进处理的文章将会定期复习/阅读。当你间隔一段时间后继续阅读一篇文章时，你将进阅读新的部分，用Alt+X将新的重要的信息摘录到不同的元素中（即Remember extract）。通常情况下，你可以使用Delete before cursor（Alt+\）删除已处理文章的部分。

决定问答材料复习和阅读材料复习时间的算法是相似的，但不完全相同。最重要的是，默认情况下，所有的复习和阅读文章都会以递增的间隔进行。在渐进阅读中，将不断有新文章加入您的收藏。未加工的材料需要与新加入的材料竞争。如果不能及时处理文章，则增加复习间隔以确保原来的材料在较低优先级。加工速度将取决于你投入的时间和材料本身的价值。那些枯燥乏味、写得不好、对你的工作或成长不太重要的文章，会占用你小撮的注意力，而且通常有很长的复习间隔。这是大量的新信息加入你的收藏和你的记忆中带来的一个不可避免的副作用。但是，时间间隔和优先级很容易调整。如果您的优先级发生变化，您可以修改处理重要文章的方式。在复习的时候，你可以不间断地阅读整篇文章，或者在较短的时间内安排复习。您可以手动更改其优先级（Alt+P）。你也可以使用搜索工具（Ctrl+F）来查找你认为你忽略了的主题的更多文章。你可以通过改变一堆文章的优先级来重新排序。您可以缩短文章的复习间隔，或者在需要时全部复习（请参阅：筛选组复习*）。

复习问题和答案（例如完形填空）的算法相当复杂，并且限制了你对复习时间的影响（参见：SuperMemo算法*）。这是为了确保你能使记忆力保持在高水平，因为记忆可能会受到手动干预的影响。但是，确定主题的复习间隔的算法要简单得多，完全由您控制。每一篇文章都有一个优先级。优先级决定了哪些文章是首先被复习的，哪些可以在您没有时间的情况下延迟复习计划。每一篇文章也被分配了一个称为A因子的数字，这个数字决定了后续复习之间的间隔增加了多少。例如，如果A因子为2，则每次复习的间隔将翻倍（乘以2）。优先级和A因子是软件自动计算的，但您也可以随时手动更改它们。优先级和A因子是根据文本的长度、处理方式、延迟或提前方式以及许多其他因素共同确定和修改的。通过按Alt+P可以更改文章的优先级和A因子。也可以使用Shift+Ctrl+↑和Shift+Ctrl+↓来增加或减少元素的优先级。请注意，与问答卡片相关联的A因子不能由用户更改，因为它们是问答卡片难度的反映，决定了最佳复习间隔的长度（请参阅：遗忘指数*）。

您可以通过手动调整复习间隔来控制文章复习的时间。使用Ctrl+J（Reschedule）或Shift+Ctrl+R（Execute repetition）来确定下一次复习的日期。Ctrl+J将增加当前间隔，而Shift+Ctrl+R将首先复习，然后设置新间隔。例如，如果当前间隔为100，并且在Reschedule中指定值3，则设置为3天内安排复习，而下一次复习日期不会更改（新间隔为103）。如果使用Execute repetition，则新间隔将为3，下一次复习日期将设置为今天。换句话说，Reschedule增加间隔（它也可以缩短间隔），Execute repeation设置间隔的长度（同时留下在学习过程中执行的复习的痕迹）。请注意，在Next repetition阶段执行的Reschedule（Ctrl+J）将首先被复习，并具有与执行Execute repetition（Shift+Ctrl+R）相同的效果。要在学习过程中延迟复习，可以在前一个复习周期进行以上二选一的操作。

在一个超负荷的渐进阅读过程中，你常常想在某一天（例如考试前）专注于某一特定的主题。为此，请阅读无价宝典：筛选组学习*。

总结
  • 使用Learn按钮处理、学习和复习所有知识
  • 问答卡片的复习由SuperMemo算法处理。对你的问答卡片进行评分，制定复习计划，并如实为他们标记上优先顺序。SuperMemo会妥善处理学习过程中的各个环节。
  • 主题/文章会以越来越长的间隔进行复习，但是，您始终可以使用Execute repetition（Shift+Ctrl+R）手动设置下一个复习日期。一定要把你最重要的文章放在高优先级。否则，它们会很快从视野中消失（这也就代表着更快遗忘）

技巧5：处理大量知识

在渐进阅读中，你可能会快速导入并产生更多的学习材料，然而你并来不及有效的处理它们。为了确保能够快速处理过载的学习材料，SuperMemo将使用优先级队列。

使用Alt+P（在元素菜单选择Priority >>Modify），可以将每个元素的优先级从0%设置为100%。注意0%对应最高优先级！

默认情况下，未完成的复习将自动从高优先级降到低优先级。这样，如果你不能完成你每天的学习，未学习的部分将会下降到较低优先级。此外，默认情况下，在工作日开始时（即，在第一次运行SuperMemo时），您前几天的未完成材料将自动推迟（同样，高优先级材料受影响最小）。

阅读有关优先级队列的文章以了解有关以下内容的更多信息：

  1. 元素的手动排序，
  2. 定义排序条件，
  3. 关闭自动排序和自动延迟等功能。
