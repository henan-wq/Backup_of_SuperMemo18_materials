Title: 2.4 其他基础技能

渐进阅读中的知识进化

SuperMemo中知识进化的三个主要原则是：

  • 降低复杂性——文章将被转换成由段落组成的集合。段落将被分解成一组独立的句子和观点，同时将缩短非必须的句子成分以降低复杂性。
  • 主动回忆——所有的信息最终将被转换为主动回忆材料，例如问答、完形填空、图片识别测试、声音识别测试等。这是为了使你的知识回忆最大化。
  • 渐进主义——所有的变化都会随着投入时间的增加而逐渐发生，与你所选材料的优先级有关，并与储存强度逐渐增加成正比。在SuperMemo学习的渐进性质将帮助您在最短的时间内获得最大的记忆效果。见：间断学习的价值

使用图片

为了获得更多的信息、记忆线索和纯粹的学习乐趣，你可以在SuperMemo中以渐进方式阅读一篇文章，并用从其内容或其他来源拍摄的有意义的图片加以说明。按Ctrl+F8选择一张图片嵌入文章。

如果您从维基百科导入，SuperMemo 16或更高版本可以用全图而不是缩略图来为一篇文章及其所有摘录和完型填空添加插图。

摘录卡片与问答卡片

在SuperMemo中，您可以看到两种基本形式的信息：

  • 摘录卡片 topics：这些通常是您想阅读的较长的文章
  • 问答卡片 items：这些通常是你需要回答的具体问题

摘录卡片和问答卡片在不同的时间以不同的方式呈现。摘录卡片保留你想学习的知识（即你想读的东西），而问答卡片保留你想记住的知识（即你已经拥有但可能忘记的知识）。

摘录卡片

SuperMemo中的摘录卡片是一篇文章、它的一部分或一个你想学习的句子。摘录卡片也可以有图片、视频、音乐等形式。与问答卡片不同，主题不会对您的知识进行测验。它们仅用于被动阅读、观看或收听。短文本摘录卡片用于生成完形填空。摘录卡片参与渐进学习过程。一旦转换成问答卡片，它们通常会被搁置（即在学习过程中被忽略）或完成（即从学习过程中完全删除）。Done!和Dismiss 必须由用户执行（即它们不是自动的）。

使用摘录卡片

在Contents中用绿色的T图标（）标记摘录卡片。摘录卡片可以是很长（整篇文章）或很短（单句话）。以下是您处理主题的方式：

  • 从顶部阅读摘录卡片
  • 如果您发现一些有趣的信息，请将其摘录（Alt+X）出来；摘录片段将形成一个新的独立摘录卡片；新摘录卡片将更短，并将以与所有其他摘录卡片相同的方式处理
  • 根据摘录卡片的优先级和可用时间，决定阅读该主题的程度（例如，如果您很匆忙，请快速中断；如果主题是最重要的，请全部阅读）
  • 如果你读完了这个主题，就执行Done! （Ctrl+Shift+Enter）；这将删除摘录卡片而不删除其生成的材料
  • 只有当主题像一个句子那么短时，才能创建完形填空（Alt+Z）
  • 下次要复习的时候再看这个题目

在较长的主题进行阅读和摘录，在非常短的主题上生成完形填空。

问答卡片

SuperMemo中的问答卡片是你想要记住的知识。它通常是一个问答形式。一个问答卡片和一个摘录卡片的主要区别在于，问答卡片主动测试你的记忆（例如，用一个问题），而摘录卡片只用于被动复习（例如，用于阅读、观看、收听等）。

概念

SuperMemo 17引入了一种新的元素类型：概念（在Contents中用橙色灯泡图标表示（））。这样的元素代表一个重要的思想或主题。多个摘录卡片和问答卡片（甚至是任务）可以链接到一个概念。链接将它们与概念所代表的思想/主题关联起来。基于概念的链接网络称为concept map。它形成了一个扩展激活的框架，这是神经回顾的基础。（It forms a skeleton for the spreading activation which underlies neural review.）

任务

除了问答卡片、摘录卡片和概念之外，您还可以在渐进学习中使用任务。任务是按价值/时间比或价值/代价比排序的工作。

有关SuperMemo中问答卡片、摘录卡片、概念和任务的全面比较，请参见：SuperMemo中的元素类型*。

超负荷阅读

当学生有比他能处理的更多的未完成的问答卡片或摘录卡片需要复习时，就会出现超负荷。很少有用户能保持住每天都复习超过200个问答卡片。当统计窗口中的未完成参数开始超过200时，可能会出现超负荷。

最好用自动延迟来处理超负荷工作。然而，一个偶然性的超负荷可以通过Postpone（延迟所有元素）或 Mercy（及时分摊所有复习任务）有效地解决。

还可以使用以下方法推迟特定摘录卡片及其所有摘录：

跳转到要处理的摘录卡片

按Ctrl+Space在浏览器中打开摘录卡片、其摘要和完形填空

在浏览器菜单上选择Process browser> >> Postpone

请注意，如果您已将部分学习材料移动到其他分支，则可能需要使用元素菜单栏上的Learning >> Locate extracts 。

自动排序和自动延迟

你可以通过以下选项，轻松地处理高优先级的学习材料：

Learn >> Sorting >> Auto-sort repetitions：在每天开始时按优先级对未完成的队列进行排序。

Learn >> Postpone >> Auto-postpone ：在每天学习开始时推迟优先级较低的未完成的复习。它确保您不会超负荷学习，并确保您最大限度地减少对高优先材料的延迟。

自动延迟总是将最重要的元素留在队列中。延期的目的是让你集中精力于高优先级材料，防止低优先级的材料成为复习计划的主体。您最有可能在学习一天后使用Postpone，而在学习日开始之前应用Auto-postpone。这就是为什么Postpone从不影响今天的学习材料，也不会推迟前几天积攒的重要材料。如果在菜单上选中了Auto-postpone，则在开始一天的学习时，将会复习当天应当复习的所有材料，以及前几天的一些未处理的最高优先级的复习材料。即使Auto-postpone增加间隔并减少低优先级材料的复习机会，它也使您在间隔复习中受益。研究表明，较长的间隔可能会自相矛盾地增加学习速度（直到某一点）。这是因为SuperMemo中的默认记忆保留率（约95%）高于每单位时间内记忆的问答卡片数量的最大值。

你可以从分类标准的默认设置开始，但是，如果你觉得你在问答卡片上没有取得足够的进展（例如，高遗忘指数），你可以减少摘录卡片的比例。如果新材料引入的太慢，可以增加摘录卡片的比例。如果你的优先级不完美，增加随机化的程度。如果您认为您错过了太多高优先级的问答卡片（请选择主菜单中的Toolkit >> Statistics >> Analysis >> Use >> Priority protection），请降低随机化。经过反复试验，你将达到最佳状态。即使你找到了自己的最佳选择，也要不断尝试不同的随机化和摘录卡片级别。这将有助于你避免在日常学习中产生的各种认知偏差。经常进行随机复习也可能会有帮助（只是对你的整体进程有一个大致的了解）。

有了Auto-sort 和 Auto-postpone，你几乎不用担心材料超负荷。每天你第一次启动SuperMemo的时候，它都会首先推迟你在前几天没有执行的复习。它将使用默认的延迟条件，您可以随时修改（Learn >> Postpone >> All elements）。在推迟了积压的复习任务之后，SuperMemo将对今天的复习任务和Auto-postpone后留下的未完成的复习任务进行排序。Auto-sort将使用前面指定的排序条件（Learn >> Sorting >> Sorting criteria）。

有了Auto-sort 和 Auto-postpone，你要学习的材料每天会根据优先级自动排序。你的学习顺序将被优化而不需要你的任何操作（即没有选择，也没有按键可以让你选择）。

超负荷提示

不管有没有Auto-postpone，你唯一可以确定的防止遗忘的方法总是一样的：完成你的重复！ 自动延迟影响除今天以外的所有日期。如果今天排定的主题优先级较低，则“自动推迟”将仅在明天延迟，并且仅在您今天不审阅时才延迟。这是为了确保低优先级主题也有机会根据排序标准中的随机化/优先级平衡来确定重复项 在“推迟”对话框中，跳过以下数量的最高优先级元素仅跳过“参数”选项卡上的跳过条件跳过的元素。如果元素不受延迟条件的保护，则不会保护它们不被延迟。不管这个参数的值是什么，您仍然可以推迟所有元素。您最好将其视为一个pro-delay参数，用于强制执行额外的延迟（而不是保护您免受额外延迟影响的反延迟参数）。这里的跳过意味着“跳过延迟保护”而不是“跳过延迟” 可以使用“延迟中模拟”来告诉您当前的延迟条件工作得如何。它忽略跳过以下数量的最高优先级元素，因为此参数不需要模拟（它将始终强制跳过所述数量的受延迟条件保护的元素）

筛选组复习

筛选组复习是对部分学习材料的复习（如考试前）。该部分可以通过搜索、Contents中的分支选择、concept group和其他筛选元素的方法来创建筛选组。复习的筛选组材料可以按其在知识树（Contents）、优先级、难度、间隔、最近度等顺序进行排序。
